# Tìm kiếm người dùng

Method **`GET`**

```shell
https://hrmadmin.herokuapp.com/api/search-user?email=<userEmail>
```

### Header

| Trường dữ liệu  | Kiểu   | Chú thích                                   |
| --------------- | ------ | ------------------------------------------- |
| `Authorization` | string | `Bearer` + [`accessToken`](../access-token.md) |
