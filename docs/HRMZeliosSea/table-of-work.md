# Bảng công

Bao gồm chi tiết của từng ngày:

- `Trạng thái`
- `Thời gian chấm công lần đầu`
- `Thời gian chấm công lần cuối cùng`
- `Tổng số giờ làm`
- `Ngày công`

## Trạng thái

- `Bình thường`
- `Nghỉ có phép`
- `Nghỉ không phép`

## Ngày công

| Tham số | Chú thích     |
| ------- | ------------- |
| `0`     | Ngày nghỉ     |
| `0.5`   | Nửa ngày công |
| `1`     | Một ngày công |
