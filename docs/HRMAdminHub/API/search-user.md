# Tìm kiếm người dùng


Method **`GET`**

```shell
https://hrmadmin.herokuapp.com/api/searchUser
```

### Tham số

| Trường dữ liệu | Kiểu   | Chú thích           |
| -------------- | ------ | ------------------- |
| `email`     | string | email người dùng    |

### Header

Tìm hiểu về cách lấy [`Access token`](sign-in.md)
