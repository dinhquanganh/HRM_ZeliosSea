# Lấy thông tin người dùng

Method **`GET`**

```shell
https://hrmadmin.herokuapp.com/api/user-info/<userId>
```

### Header

| Trường dữ liệu  | Kiểu   | Chú thích                                   |
| --------------- | ------ | ------------------------------------------- |
| `Authorization` | string | `Bearer` + [`accessToken`](../access-token.md) |
