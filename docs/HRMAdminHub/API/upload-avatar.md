# Thay ảnh đại diện

Change avatar for user.

Method **`POST`**

```shell
https://hrmadmin.herokuapp.com/upload/avatar/:id
```

### Parameter

| Trường dữ liệu | Kiểu | Chú thích         |
| -------------- | ---- | ----------------- |
| `image`        | file | image file avatar |
