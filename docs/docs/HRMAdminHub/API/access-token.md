# Access token

Method **`POST`**

```shell
https://hrmadmin.herokuapp.com/api/token
```

### Tham số

| Trường dữ liệu | Kiểu   | Chú thích           |
| -------------- | ------ | ------------------- |
| `username`     | string | email người dùng    |
| `password`     | string | mật khẩu người dùng |
