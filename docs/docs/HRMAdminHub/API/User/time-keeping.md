# Chấm công
Method **`PUT`**

```shell
https://hrmadmin.herokuapp.com/api/location/<userId>?lat=<latitude>&lon=<longitude>
```

### Header

| Trường dữ liệu  | Kiểu   | Chú thích                                   |
| --------------- | ------ | ------------------------------------------- |
| `Authorization` | string | `Bearer` + [`accessToken`](../access-token.md) |
