# Lấy thông tin một vấn đề trong dự án

Method **`GET`**

```shell
https://hrmadmin.herokuapp.com/api/projects/<pid>/issues/<iid>
```

| Trường dữ liệu  | Kiểu   | Chú thích                                   |
| --------------- | ------ | ------------------------------------------- |
| `Authorization` | string | `Bearer` + [`accessToken`](../access-token.md) |
