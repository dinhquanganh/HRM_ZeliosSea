# Lấy thông tin vấn đề

Method **`GET`**

```shell
https://hrmadmin.herokuapp.com/api/issueInfo/<issueId>
```

### Header

| Trường dữ liệu  | Kiểu   | Chú thích                                   |
| --------------- | ------ | ------------------------------------------- |
| `Authorization` | string | `Bearer` + [`accessToken`](../access-token.md) |
