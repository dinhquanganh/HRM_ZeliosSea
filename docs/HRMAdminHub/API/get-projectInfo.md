# Lấy thông tin dự án

Method **`GET`**

```shell
https://hrmadmin.herokuapp.com/api/projectInfo/:id
```


### Header

| Trường dữ liệu  | Kiểu   | Chú thích                                   |
| --------------- | ------ | ------------------------------------------- |
| `Authorization` | string | `Bearer` + [`accessToken`](access-token.md) |
